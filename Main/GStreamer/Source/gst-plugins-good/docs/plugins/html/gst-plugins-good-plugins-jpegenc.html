Description

Encodes jpeg images.

Example launch line

1
 
gst-launch videotestsrc num-buffers=50 ! video/x-raw-yuv, framerate='(fraction)'5/1 ! jpegenc ! avimux ! filesink location=mjpeg.avi
a pipeline to mux 5 JPEG frames per second into a 10 sec. long motion jpeg avi.
